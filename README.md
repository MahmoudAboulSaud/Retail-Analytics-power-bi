# Retail Sales Analysis Dashboard (Power BI + SQL Server)

## 📌 Project Overview
This is an end-to-end data analysis project that transforms raw retail data into actionable business insights using SQL Server and Power BI.

The project includes data cleaning, data modeling, SQL-based analysis, and interactive dashboards.

## 🎯 Objectives
- Analyze retail sales performance
- Identify top products, customers, and regions
- Track KPIs such as revenue, profit, and returns
- Build interactive dashboards for decision-making

## 🛠 Tools & Technologies
- SQL Server
- Power BI
- Excel
- Data Visualization
- Data Modeling

## 🔄 Project Workflow
1. Data Collection
2. Data Cleaning (SQL)
3. Data Transformation
4. Data Modeling
5. Dashboard Development (Power BI)
6. Insights Generation

## 📊 Dashboard Features
